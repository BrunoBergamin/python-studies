# Estudos de Python

Este repositorio foi organizado para acompanhar meus estudos em Python de forma simples, progressiva e pratica. A proposta e estudar como um aluno iniciante ou intermediario: entendendo a teoria e logo em seguida testando em pequenos arquivos.

Aqui eu pratico desde os fundamentos da linguagem ate os primeiros conceitos de orientacao a objetos.

## Objetivo do repositorio

- Aprender Python na pratica
- Treinar logica de programacao
- Organizar os estudos por assunto
- Centralizar toda a documentacao em um unico README
- Criar exemplos simples para revisar depois
- Evoluir dos fundamentos para temas mais estruturados, como POO

Nesta etapa eu pratico:

- Tipos primitivos como `str`, `int`, `float` e `bool`

- Operadores logicos como `and` e `or`
- Repeticao com `for` e `while`
- Criacao de funcoes com `def`
- Retorno de valores com `return`

## Descricao da pasta de orientacao a objetos

### `python-fundamenos/orientacao-objetos/exemplo_classe_aluno.py`
Exemplo basico de classe `Aluno`, com atributos como nome, idade e curso. Mostra como criar objetos e chamar metodos.

### `python-fundamenos/orientacao-objetos/exemplo_carro.py`
Exemplo de classe `Carro` para praticar atributos de estado, como velocidade, e metodos como acelerar e frear.

### `python-fundamenos/orientacao-objetos/exemplo_conta_bancaria.py`
Exemplo de classe `ContaBancaria`, trabalhando metodos como depositar, sacar e mostrar saldo.

### `python-fundamenos/orientacao-objetos/exemplo_heranca.py`
Exemplo simples de heranca com a classe `Pessoa` e a classe `Aluno`, para entender reaproveitamento de codigo.

## Ideia de estudo como aluno

Minha organizacao neste repositorio segue uma logica simples:

1. Primeiro eu aprendo a sintaxe basica.
2. Depois eu estudo os tipos primitivos.
3. Em seguida eu pratico condicionais e repeticao.
4. Depois eu treino listas, dicionarios e funcoes.
5. Por fim, comeco a estudar orientacao a objetos com exemplos pequenos e claros.

Isso ajuda a nao misturar muitos conceitos ao mesmo tempo e deixa a evolucao mais facil de acompanhar.
